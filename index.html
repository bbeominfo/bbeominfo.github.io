01 · Mar – May 2025
Active Learning-Guided Protein Mutation Effect Prediction
ML Developer · MLP, ESM-1v, Active Learning · Georgia Tech
Problem Statement

Predicting the functional impact of protein mutations is critical for drug development and precision medicine. This project used Deep Mutational Scanning (DMS) scores as a fitness proxy to build ML models that predict how single amino acid mutations affect protein function.

Methods
Feature Extraction — Combined 1280-dim mean-pooled ESM-1v embeddings with 6 biochemical features (aromaticity, hydrophobicity, isoelectric point, helix/turn/sheet fraction) → 1286-dim input vector
MLP Ensemble (Final) — 4-layer MLP (1286→512→256→64→1) with LayerNorm + LeakyReLU + Dropout(0.4), trained with AdamW over 40 epochs, ensembled across 3 random seeds
Benchmarked architectures — CNN (ESM-2 + RF feature selection), RNN (one-hot encoding), ESM fine-tuning (partial, layers 31–32)
Active Learning — 3 query strategies: MC Dropout uncertainty, ESM log-probability difference (ΔlogP), and UCB-based Gaussian Process sampling
Results
0.49
Best validation Spearman (MLP · full-seq mean + biochem)
0.27
Leaderboard Spearman (MLP vs CNN 0.26 / RNN 0.07)
0.18
ESM fine-tuning score — frozen embeddings outperformed

MLP ensemble outperformed CNN, RNN, and ESM fine-tuning. In low-data settings, freezing PLM and using a downstream MLP proved more effective than partial fine-tuning — which required 40–60GB GPU memory with lower performance.

My Contributions
Designed and implemented the full MLP pipeline including feature engineering, architecture design, and ensemble strategy
Implemented parameter-efficient ESM fine-tuning and led comparative analysis across all architectures
Designed all 3 active learning query strategies (MC Dropout, ΔlogP, UCB-GP)
Wrote the final report and led experimental documentation
Future Work
Incorporate 3D structural features (SASA, PDB-based) for richer mutation context
Explore LSTM/GRU-based architectures with embedding layers instead of one-hot encoding
Apply larger-scale ESM-2 models with more efficient fine-tuning (LoRA, adapter layers)

02 · Spring 2025 · CX4803 ML in Computational Biology
Drug-Disease Association Modeling via Heterogeneous GNNs
Deep Learning Developer · Georgia Tech
Problem Statement

Traditional drug discovery is lengthy and costly. This project aimed to predict drug-disease associations using heterogeneous Graph Neural Networks on the Hetionet biomedical knowledge graph — enabling drug repurposing by identifying existing compounds that may treat unknown diseases.

Methods — 4 Model Architectures Benchmarked
Node2Vec + GCN — Structural baseline; biased random walk embeddings fed into a homogeneous 2-layer GCN. Highest internal AUC (0.99) but strong overfitting on external data.
Sentence Transformer + HeteroConv (Final) — Node names embedded via all-MiniLM-L6-v2 (384-dim) into 2-layer SAGEConv with 13 meta-relations. Zero-shot capable for unseen diseases.
Node2Vec + R-GCN — 5-layer relational GCN with relation-specific parameters across 8 meta-edges. AUC 0.96 on Hetionet.
Node2Vec + HGT — 2-layer Heterogeneous Graph Transformer with per-node-type and per-relation multi-head attention. AUC 0.77.
Results
0.89
AUC on Hetionet (final model · Sentence Transformer + HeteroConv)
0.78
F1-score on Hetionet test set
0.63
AUC on external HINGRL-B dataset (647 validated pairs)
4
GNN architectures compared

The final model (Sentence Transformer + HeteroConv) was selected over the higher-AUC GCN due to superior diversity in Top-N drug recommendations — the GCN consistently recommended the same highly-connected compounds across all diseases (degree centrality bias).

Key Findings
Simple neighborhood aggregation (SAGEConv) outperformed attention-based HGT in generalization, suggesting structural bias from Node2Vec limits complex architectures.
Semantic text embeddings enabled zero-shot prediction for unseen diseases — a critical advantage over purely structural approaches.
High internal AUC does not guarantee real-world utility; qualitative diversity of recommendations is equally important in drug discovery.
Future Work
Incorporate multi-modal node embeddings: SMILES for chemical structure, gene ontology, BERT from biomedical literature
Apply degree normalization and biological filtering to reduce hub-compound bias
Extend to protein-structure data (PDB) for richer drug-target representations
